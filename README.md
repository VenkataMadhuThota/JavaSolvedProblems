# Java Coding Practice

A collection of Java problems solved from scratch, covering core concepts, data structures, algorithms, and problem-solving patterns — built to strengthen fundamentals and prepare for technical interviews.

---

## Problems Solved

This repository includes solutions on topics such as:

- String manipulation and pattern problems
- Array and number-based logic
- OOP concepts and design patterns
- Data structures (linked lists, stacks)
- Sorting and searching algorithms

---

## How to Run

Each file is a standalone Java program with a `main` method.

```bash
javac FileName.java
java FileName
```

Or open the project in IntelliJ IDEA, Eclipse, or Spring Tool Suite and run any file directly.

**Requirements:** JDK 8 or above
